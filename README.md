# Icons8 WPF Framework

Windows 8 UI framework by Icons8 is a free set of UI controls that can be used for designing better user interfaces for Windows.

**Set includes:**
- Off-canvas menu 
- Grid with sorting 
- Toolbar 
- One-level Ribbon 
- Search 
- Form elements 
- Input box 
- Buttons 
- Radio button, checkbox, combo box etc.
